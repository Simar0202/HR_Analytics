HR Analytics Dashboard - Power BI
Project Overview

Developed an HR Analytics Dashboard in Power BI to analyze employee attrition trends and identify factors contributing to workforce turnover.

-----------Data Cleaning
Checked dataset for missing values.
Retained null values where removal would result in significant data loss.
Removed duplicate records.
Standardized categorical values for consistency (e.g., TravelRarely → Travel_Rarely).
------------Tools Used
Power BI
Power Query
DAX
CSV Dataset
--------------KPIs
Total Employees
Active Employees
Attrition Count
Attrition Rate
Average Age
Average Salary
Average Years at Company
--------------Dashboard Insights
Employees aged 26–35 showed the highest attrition.
Sales Executive and Laboratory Technician roles experienced the highest attrition counts.
Lower salary bands showed higher employee attrition.
Life Sciences and Medical education backgrounds contributed significantly to attrition.
Research & Development and Sales departments recorded the highest attrition levels.
